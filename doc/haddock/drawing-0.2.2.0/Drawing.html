translated :: Double -> Double -> Drawing -> Drawing Source #

A drawing translated by the given x and y coordinates.

translated dx dy = transform (translation (dx, dy))

scaled :: Double -> Double -> Drawing -> Drawing Source #

A drawing scaled by the given x and y factors. Scaling by a negative factor also reflects across that axis.

scaled sx sy = transform (scaling sx sy)

dilated :: Double -> Drawing -> Drawing Source #

A drawing scaled uniformly by the given factor. Dilating by a negative factor also reflects across the origin.

dilated s = scaled s s

rotated :: Double -> Drawing -> Drawing Source #

A drawing rotated by this angle about the origin.

Angles are in radians.

rotated a = transform (rotation a)

xSkewed :: Double -> Drawing -> Drawing Source #

xSkewed a = transform (skewX a)

ySkewed :: Double -> Drawing -> Drawing Source #

ySkewed a = transform (skewY a)

Affine transformations

data Transform Source #

An affine transformation.

An affine transformation is any transformation that preserves collinearity (i.e., all points lying on a line initially still lie on a line after transformation) and ratios of distances (e.g., the midpoint of a line segment remains the midpoint after transformation). An affine transformation is also called an affinity.
